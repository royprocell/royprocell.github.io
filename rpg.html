Pathfinder

Introduction

Pathfinder is a role-playing game project that started in 2015. Pathfinder utilizes 2D graphics and is written in Java. While initially following a tutorial, I decided to move on from that tutorial and started modifying and creating new features to make a complete Java 2D game engine. When finished with Pathfinder, I plan to make a platforming game using this game engine.

In December 2015, I created a Github Repository for Pathfinder. Click here to view the code.

Features include:

  • Graphical interface
  • Custom map loading
  • Inventory and Bank (in progress)
  • Quests (in progress)
  • Monsters (in progress)
  • NPCs and dialogue (in progress)
